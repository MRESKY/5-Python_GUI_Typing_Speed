# Python GUI Typing Speed Test

A desktop application built with Python and Tkinter that allows users to test and improve their typing speed through an interactive GUI.

## Features

- **Typing Speed Test**: Measure your typing speed in words per minute (WPM).
- **Accuracy Tracking**: Keep track of your typing accuracy during tests.
- **Customizable Texts**: Choose from different text samples to type.
- **Timer Functionality**: Set time limits for typing tests.
- **User-Friendly Interface**: Simple and intuitive GUI for ease of use.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/your-repo.git
   cd your-repo
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
    python -m gui.main_window
   ```

## Usage

- Launch the application using the command above.
- Select a text sample and start the typing test.
- Monitor your WPM and accuracy in real-time.
- After the test, review your performance statistics.
